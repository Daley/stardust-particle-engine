Classpublic class PV3DDisplayObjectClass
InheritancePV3DDisplayObjectClass Initializer3D Initializer StardustElement

Assigns a Particle object to the target properties of a particle, and uses a display object along with a MovieParticleMaterial object to create particle material. This information can be visualized by PV3DDisplayObjectRenderer.
