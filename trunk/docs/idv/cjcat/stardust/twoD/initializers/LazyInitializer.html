Classpublic class LazyInitializer
InheritanceLazyInitializer CompositeInitializer Initializer StardustElement

This is a wrapper class of several common and useful initializers. As it's name suggests, you can use this class whenever you're too lazy to create initializers one by one. You don't directly control these component initializers; instead, you use the abstract properties provided by this class.

This composite initializer is consisted of the folling actions:

  • DisplayObjectClass
  • Position
  • Rotation
  • Velocity
  • Omega
  • Scale
  • Mass
  • Mask
  • CollisionRadius
  • Alpha
