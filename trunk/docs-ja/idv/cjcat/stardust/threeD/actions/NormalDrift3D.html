Property Detail
masslessproperty
public var massless:Boolean

パーティクルの加速度を、質量で割った値で適用するかどうかを指定します、デフォルトは true です. true がセットされていると、すべてのパーティクルに等しい加速度を適用します.

maxproperty 
public var max:Number

加速度は、 -max から max までの範囲になります.

randomproperty 
random:Random

ランダムオブジェクトは、加速度の範囲内 [-max, max] の乱数を生成するために使用されます、デフォルトは UniformRandom です. ランダムオブジェクトの範囲を設定する必要はありません. 範囲はその都度、自動的にランダムに作成されます.
