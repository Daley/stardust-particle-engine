Classpublic class SnapshotRestore3D
InheritanceSnapshotRestore3D Action3D Action StardustElement Object

Snapshot クラスによって、以前に取られた「スナップショット」からパーティクルの状態を復元します. また、復元するまでのイージング曲線とそれの適用時間を指定できます.

デフォルトの優先順位は -7 です.
