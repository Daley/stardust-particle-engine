MutualGravity3D アクションを使用すると、パーティクル同士が引き付き合うようになります.

epsilonproperty 
public var epsilon:Number

2 つのパーティクル間の距離がこの値より離れていた場合は、この値以下の距離に収められます. このプロパティはシミュレーションを暴走させないためにあります、デフォルトは 1 です

masslessproperty 
public var massless:Boolean

すべてのパーティクルを同じ質量と見なすかどうかを指定します、デフォルトは true です.

false にセットされていると、パーティクルの質量は考慮されます. ( 重いパーティクルが軽いパーティクルよりも移動しないようになります )
