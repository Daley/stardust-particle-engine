RandomDrift3D アクションを使用すると、パーティクルの速度を不規則に加速させることができます.

デフォルトの優先順位は -3 です.

masslessproperty
public var massless:Boolean

パーティクルの加速度を、質量で割った値で適用するかどうかを指定します、デフォルトは true です. true がセットされていると、すべてのパーティクルに等しい加速度を適用します.

randomXproperty 
randomX:Random

ランダムオブジェクトは、加速度の Y 成分の範囲内 [-maxY, maxY] の乱数を生成するために使用されます、デフォルトは UniformRandom です. ランダムオブジェクトの範囲を設定する必要はありません. 範囲はその都度、自動的にランダムに作成されます.
