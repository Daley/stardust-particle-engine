Classpublic class PV3DParticleHandler
InheritancePV3DParticleHandler ParticleHandler StardustElement Object

PV3DParticleHandler クラスは、 Particles オブジェクトに particle オブジェクトを加え、 Particles オブジェクトから死んでいるパーティクルを取り除き、 Particle オブジェクトの x 、 y 、 z 、 rrotationZ 、 および size プロパティを更新するハンドラーです.
