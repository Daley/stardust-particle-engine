Classpublic class DisplayObjectHandler3D
InheritanceDisplayObjectHandler3D ParticleHandler StardustElement Object

DisplayObjectHandler3D は、対象のコンテナーの表示リストに、表示オブジェクトであるパーティクルを追加して、 表示リストから死んでいるパーティクルを取り除き、 表示オブジェクトの x 、 y 、 rotation 、 scaleX 、 scaleY 、および alpha プロパティを変更し、 Z ソートおよび透視投影を表示オブジェクトに適用するハンドラーです.
