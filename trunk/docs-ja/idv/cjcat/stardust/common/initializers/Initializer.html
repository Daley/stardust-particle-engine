Classpublic class Initializer

Implements InfoRecycler
Subclasses Alpha, CollisionRadius, Color, CompositeInitializer, Initializer2D, Initializer3D, Life, Mask, Mass, Scale, SwitchInitializer

イニシャライザーは、パーティクルの発生した瞬間 (すなわち、初期設定) にパーティクルのプロパティを変更するのに使用します.

エミッタかパーティクルファクトリにイニシャライザーを関連付けることができます.

デフォルトの優先順位は 0 です.

priorityproperty 
priority:int

イニシャライザーは優先順位に従いソートされます.

いくつかのイニシャライザーは、先に初期化すると、他のイニシャライザーに依存する可能性があるため、このプロパティは重要になります. イニシャライザーの優先順位を変更することは出来ますが、デフォルト値を使用することが推奨されています.
