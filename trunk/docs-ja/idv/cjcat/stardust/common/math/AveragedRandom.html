Classpublic class AveragedRandom
InheritanceAveragedRandom Random StardustElement Object

AveragedRandom クラスは、 Random オブジェクトの random() メソッドを複数回呼び出し、平均値を求めます.

通常、sampleCount が大きければ大きいほど、結果がより分散されます.
