Property Detail
activeproperty
public var active:Boolean

エミッターがアクティブであるかどうか、デフォルトは true です.

エミッタがアクティブであるなら、クロックに従って各ステップ時にパーティクルを作成します. エミッタがアクティブでなくても、既存のパーティクルのシミュレーションは、各ステップ時に行われるので注意してください.

particleCollectionTypeproperty 
particleCollectionType:int

内部的にエミッターによって使用されるコレクションを指定します. LinkedList と配列の2つのオプションがあります. パーティクルの反復処理や取り除くことは、 LinkedList の方が通常より速いのですが、ソートは配列の方がより速く行うことができます. デフォルトでは、 LinkedList が使用されています. パーティクルをソートする必要があるなら、配列に切り替えてください.

このプロパティに割り当てられる有効な値は ParticleCollectionType.LINKED_LIST と ParticleCollectionType.ARRAY の 2 つです.

step()method 
public final function step(time:Number = 1):void

このメソッドは、エミッターのシミュレーションにおけるメインループです.

シミュレーションを続けるために、このメソッドは継続的に呼ぶべきです. Event.ENTER_FRAME イベントや TimerEvent.TIMER イベントを介してこのメソッドを呼び出すことを推奨します.

Parameters

time:Number (default = 1) — シミュレーションのシングルステップの時間間隔. 例えば、このパラメーターを倍にすると、シミュレーションは 2 倍の速度になります.
