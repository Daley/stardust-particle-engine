アクションは、継続的にパーティクルのプロパティを変更するのに使用します.

アクションはエミッターに関連付けされています. 各 Emitter.step() メソッド呼び出し時に、 エミッターとパーティクルをパラメーターとして update() メソッドに渡されます.このメソッドはパーティクルのプロパティを更新します. ( 速度に従ってパーティクルの位置を変えたり、重力に基づきパーティクルの速度を変更するなど. )

デフォルトの優先順位は 0 です.

maskproperty 
mask:int

このアクションのマスクとパーティクルのマスクで AND 演算を行い、ゼロではない場合にのみ、パーティクルにアクションの効果を適用します. デフォルト値は 1 です.

このプロパティは、特定のパーティクルにアクションの効果を適用させたくないときに使用します.

priorityproperty 
priority:int

関連付けされたエミッターによって、アクションは優先順位に従いソートされます.

これは重要です、 最初に、速度に応じてパーティクルの位置を変更しても意味をなしません. それは、その後、重力フィールドによって速度が変更されるためです. アクションの優先順位を変更することは出来ますが、デフォルト値を使用することが推奨されています.
