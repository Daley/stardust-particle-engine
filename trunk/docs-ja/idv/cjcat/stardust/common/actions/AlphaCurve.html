Packageidv.cjcat.stardust.common.actions
Classpublic class AlphaCurve
InheritanceAlphaCurve Action StardustElement Object

AlphaCurve アクションを使用すると、パーティクルのライフに従ってパーティクルのアルファ値を変更することができます.

アルファトランジションは、Robert Penner によって設計されたイージング関数を使用しています. これらの関数は idv.cjcat.stardust.common.easing パッケージ内にあります.
