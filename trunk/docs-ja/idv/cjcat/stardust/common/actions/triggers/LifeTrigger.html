Classpublic class LifeTrigger
InheritanceLifeTrigger ActionTrigger CompositeAction Action StardustElement Object

LifeTrigger クラスは、パーティクルのライフが triggerEvery プロパティの倍数の場合に引き起こされるトリガーです.

例えば、triggerEvery プロパティが 2 であり、エミッターの stepTimeInterval プロパティが 1 なら、 このトリガーは Emitter.step() を 2 度呼び出す度に実行されます.

デフォルトの優先順位は -5 です.
