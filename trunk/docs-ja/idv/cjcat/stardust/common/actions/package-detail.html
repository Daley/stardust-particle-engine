 ClassDescription
 Action アクションは、継続的にパーティクルのプロパティを変更するのに使用します.
 ActionCollection ActionCollection クラスは ActionCollector インタフェースを実装するクラスによって内部的に使用されます.
 ActionPriority ActionPriority クラスのサブクラスは、同じアクションの優先度を設定することが出来ます.
 Age Age アクションを使用すると、パーティクルのライフを減少させることができます.
 AlphaCurve AlphaCurve アクションを使用すると、パーティクルのライフに従ってパーティクルのアルファ値を変更することができます.
 CommonActionPriority CommonActionPriority クラスは、 2D 、 3D 共に汎用的に使えるいくつかのアクションの優先度を定義します.
 CompositeAction CompositeAction クラスは、アクションのグループです.
 DeathLife DeathLife アクションを使用すると、ライフがゼロのパーティクルを死亡したと扱うようになります.
 Die Die アクションを使用すると、パーティクルを即座に死亡したと扱うようになります.
 ScaleCurve ScaleCurve アクションを使用すると、パーティクルのライフに従ってパーティクルのスケールを変更することができます.
