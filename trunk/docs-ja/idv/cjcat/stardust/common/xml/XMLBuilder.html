InheritanceXMLBuilder Object

XMLBuilder は Stardust 要素から XML を作成して、既存の XML データーから要素を再現することができます.

あらゆる StardustElement オブジェクトは StardustElement.toXML() メソッドによって XML を作成することができます. それらは、既存の XML データーから StardustElement.parseXML() メソッドで再現することができます.

registerClass()method 
public function registerClass(elementClass:Class):void

StardustElement のカスタムサブクラスで XMLBuilder を使用するためには、最初に、クラスと XML タグ名を決める必要があります.

例えば、MyAction クラスを "HelloWorld" という XML タグで登録すると、 XMLBuilder は、 "HelloWorld" タグが XML で現れたとき、 MyAction クラスを参照していることが分かります. Stardust エンジンのすべてのデフォルトクラスはすでに登録されています.
