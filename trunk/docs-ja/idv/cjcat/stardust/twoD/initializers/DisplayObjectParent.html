Classpublic class DisplayObjectParent
InheritanceDisplayObjectParent Initializer2D Initializer StardustElement Object

DisplayObjectParent イニシャライザーを使用すると、パーティクルが DisplayObjectClass イニシャライザーによって初期化される場合に、 パーティクルの親を特定の表示オブジェクトコンテナに指定できるようになります.

デフォルトの優先順位は -100 です.
