Classpublic class BitmapBurster
InheritanceBitmapBurster Burster2D Burster Object

BitmapBurster クラスは、ビットマップを小さな長方形に分割し、それを target プロパティに参照したパーティクルを、一気に作成するバスターです.

作成時のパーティクルは、敷き詰めて配置され、全体で完全なビットマップを構成します.

このバースターは内部的にパーティクルの target プロパティを Bitmap オブジェクトに変更するため、 原則的に target プロパティを設定するイニシャライザーを加えても何も起こりません.
