Classpublic class PointUniburster
InheritancePointUniburster Burster2D Burster Object

PointUniburster クラスは、単一の位置から、均一に分布された角度でパーティクルを拡散して作成するバースターです.

このバースターは内部的にパーティクルの速度を変更するため、 原則的に速度を変更するイニシャライザーを加えても何も起こりません.
