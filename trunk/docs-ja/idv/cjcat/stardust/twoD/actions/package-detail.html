 Action2DPriority Action2DPriority は、いくつかの 2D アクションの優先度を定義します.

 Explode Explode アクションを使用すると、伝播の方法に沿った加速をパーティクルに適用する、一つの点から広がる衝撃波を引き起こすことができます.
 FollowWaypoints FollowWaypoints アクションを使用すると、パーティクルが一連の通過点を通って行くようになります.

 LazyAction LazyAction クラスは、汎用的で役に立つ様々なアクションのラッパークラスです.
 Move Move アクションを使用すると、速度に応じてパーティクルの位置を変更することができます.
 MutualAction [抽象クラス] 衝突や相互重力の重力などの、相互作用するアクションのための抽象クラスです.
 MutualGravity MutualGravity アクションを使用すると、パーティクル同士が引き付き合うようになります.
 NormalDrift NormalDrift アクションを使用すると、パーティクルの速度を垂直方向に加速させることができます.
 Oriented Oriented アクションを使用すると、パーティクルが速度に合わせて回転するようになります.
 RandomDrift RandomDrift アクションを使用すると、パーティクルの速度を不規則に加速させることができます.
 ReorderDisplayObject ReorderDisplayObject アクションは、表示リスト内の表示オブジェクトの順序を並べ替えます.

 SnapshotRestore Snapshot クラスによって、以前に取られた「スナップショット」からパーティクルの状態を復元します.
 SnapshotRestoreFlag

 SpeedLimit SpeedLimit アクションを使用すると、パーティクルの最高速度を制限することができます.
 Spin Spin アクションを使用すると、オメガ ( 角速度 ) に応じてパーティクルを回転させることができます.
