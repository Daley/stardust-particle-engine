Classpublic class Spawn
InheritanceSpawn Action2D Action StardustElement Object
Implements InitializerCollector

Spawn アクションを使用すると、既存のパーティクルの位置に、新しいパーティクルを生み出すことができます.

このアクションは花火やロケットの光跡などのエフェクトに使用することが出来ます.

新しく生み出されたパーティクルは、生み出させる過程で、 addInitializer() メソッドを使用して追加されたイニシャライザーによって初期化されます
